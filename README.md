# Backend - YOUR BUDDY

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- Multer (for file uploads)

## Features
- REST API for authentication, resources, and admin actions
- Local file storage (see `uploads/` folder)

## Setup
1. Install dependencies:
   ```
   npm install
   ```
2. Create a `.env` file (see `.env.example`).
3. Start the server:
   ```
   npm start
   ```
